# 银行业务管理系统项目文档
> 项目简介：满足银行对高风险业务进行登记，风险评估的业务电子化、流程化的需求

+ 项目技术选型（前后端分离开发）:
	+ 前端：HTML、CSS、JavaScript、Layui UI组件库
	+ 后端：Python、Django
	+ 数据库：MySql
	+ 服务端部署：Ubuntu、nginx、uwsgi
